Rydex is a comprehensive, full-stack ride-sharing application designed for real-time performance and scalability. The architecture is split into a modern Next.js frontend and a robust Node.js (Express & Socket.io) backend.

Core Features:

Real-Time Tracking & Dispatch: Low-latency bidirectional communication using Socket.io and interactive maps powered by Leaflet.
In-App Communication: Audio/video capabilities powered by ZEGOCLOUD.
Secure Payments: Integrated with Stripe and Razorpay for reliable transaction processing.
Authentication & Data: Secure user authentication managed by NextAuth.js and bcrypt, with data persistence handled by MongoDB and Mongoose.
